Overview

3/29/2023

For the past 5 months I have been slowly learning how to code using Javascript, HTML, and CSS. I started with Khan Academy on November 10th, 2022 and began mixing in some Sololearn lessons slowly around 1-2 months after I started on Khan Academy.

I alternate between sites because each provide different approaches to learning to code. I would highly recommend starting with Khan Academy and Sololearn before progressing to the EdX portion. It can be very overwhelming if you don't already know some basics first. Something I also highly recommend is to start with HTML because to me it was far easier to start thinking in code because it was very simple to pick up.

Harvard-CS50

Something I truly appreciate with this course is the fact they teach you how to think in code. It was something I had struggled with when I first started coding and with that struggle I felt very defeated. I needed a way to understand how code works together but I mostly needed to understand why binary is so important. I needed to fully understand how computers speak to one another for me to finally understand how the code works together. It was the missing peice for me.
